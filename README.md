# Power BI Sales Dashboard – Motorola Mobile Analysis

This project presents an interactive **Power BI dashboard** designed to analyze sales performance, customer behavior, and transaction insights for Motorola mobile products across different regions.

# Overview

The dashboard provides a comprehensive view of key business metrics, enabling data-driven decision-making through intuitive visualizations and real-time insights.

# Key Features:

* Sales Performance Metrics

  * Total Sales: 65M+
  * Total Quantity Sold: 2K units
  * Total Transactions: 322
  * Average Sales Value: 40K

* Geographical Analysis

  * Map visualization showing **city-wise sales distribution** across India
  * Helps identify high-performing regions

* Time-Based Trends

  * Daily sales quantity trends
  * Sales variation across days of the week

* Product Insights

  * Sales comparison across different mobile models (e.g., Galaxy Note 20, OnePlus Nord, Vivo Y51)
  * Brand-wise sales and quantity breakdown

* Customer Behavior

  * Customer ratings distribution
  * Helps evaluate product satisfaction

* Payment Analysis

  * Distribution of transactions by payment methods:

    * UPI
    * Cash
    * Credit Card
    * Debit Card

* Interactive Filters

  * Month-wise filtering (January–December) for dynamic data exploration

# Purpose:

This dashboard is built to:

* Monitor overall sales performance
* Identify trends and patterns
* Understand customer preferences
* Support strategic business decisions

* Power BI
